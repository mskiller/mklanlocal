# 1. 🧠 Workflow Extraction from Images

## 1.1 Objective

Automatically extract structured workflows (nodes, steps, connections) from images such as:

* Flowcharts
* UI screenshots (e.g., ComfyUI, Node-based tools)
* Diagrams

## 1.2 Backend Implementation

### New Module

`services/workflow_extractor.py`

### Responsibilities

* Image preprocessing (resize, contrast)
* OCR extraction (Tesseract / PaddleOCR)
* Layout parsing (bounding boxes)
* Node detection (basic heuristics or ML later)
* Edge inference (spatial relationships)

### Output Schema (NEW)

```json
{
  "nodes": [
    { "id": "1", "label": "Load Image", "type": "input", "position": { "x": 10, "y": 20 } }
  ],
  "edges": [
    { "from": "1", "to": "2" }
  ],
  "raw_text": "...",
  "confidence": 0.82
}
```

### Database Changes

Add to `assets` table:

* `workflow_json` (JSONB)
* `workflow_extracted_at` (timestamp)
* `workflow_version` (string)

Migration: `0007_workflow_extraction.py`

---

## 1.3 API Endpoints

### POST `/assets/{id}/extract-workflow`

* Triggers extraction
* Stores result

### GET `/assets/{id}/workflow`

* Returns structured workflow JSON

### GET `/assets/{id}/workflow/download`

* Returns downloadable JSON file
